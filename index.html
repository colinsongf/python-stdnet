Python Stdnet¶

Object data mapper and advanced query manager for redis.

Requirements¶

  • redis-py for redis backend. Redis 2.6 or above.
  • Optional cython for ultra-fast C-extensions. Recommended.
  • Optional pymongo for mongo db backend (pre-alpha).
  • Optional pulsar for asynchronous database connection.

Python Stdnet

Stdnet is an object data mapper and advanced query manager for non relational databases. Designed to be fast, memory efficient and highly customisable.
